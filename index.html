Futoshiki Game

Objective:

  • Implementation of a Futoshiki CSP Solver with two constraint propagators (Forward Checking and Generalized Arc Consistency) and two CSP models.
  • Coding Language: Python

Propagators:

  • Forward Checking (FC) for pruning constraints with exactly one uninstantiated variable.
  • Generalized Arc Consistency (GAC) for enforcing constraints.
  • Minimum Remaining Values (MRV) heuristic for variable ordering.

Futoshiki Models:

  • Model 1: Uses binary not-equal constraints for row, column, and inequality constraints.
  • Model 2: Uses n-ary all-different constraints for row and column constraints, along with binary inequality constraints.

QuadTree Image Compression Decompression

  • Language: Python

  • Framework: Custom QuadTree Implementation

  • Lossy Compression with QuadTree

  • File I/O for BMP and QDT Formats
